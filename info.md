# Kwatts pour Home Assistant

<p align="center">
  <img src="https://raw.githubusercontent.com/daxharry/kwatts/main/icon.png" alt="Kwatts" width="128" height="128">
</p>

Intégration des données **Kwatts** : couleur du jour, prix estimé, conseil et indicateur de jour favorable pour optimiser votre consommation d'énergie solaire.

## Entités créées

- 🟢🟠🔴 **Couleur du jour**
- 🔢 **Code du jour**
- 💶 **Prix estimé (€/MWh)**
- 💬 **Conseil du jour**
- 📅 **Date**
- ☀️ **Jour favorable** *(binary sensor)*

## Clé API

Obtenez votre clé sur [apps.kwatts.fr/advices](https://apps.kwatts.fr/advices)
